# AURA — Fashion Shopping Website

A stylish, multi-page static shopping website built with pure HTML & CSS.

## Pages
- **Home** (`index.html`) — Hero, categories, featured products, promo banner
- **Products** (`shop-products.html`) — Filter sidebar, product grid, pagination
- **Cart** (`shop-cart.html`) — Cart items, quantity controls, order summary

## Features
- Responsive layout
- Hover animations
- Product filtering UI
- Checkout flow steps
- Fully linked between pages
